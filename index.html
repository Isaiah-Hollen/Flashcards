Coding Flashcards

This website was created to get a class project done, but once I started the actual project which was supposed to be a couple of flashcards I had an idea. My idea was "What if I make flashcards I can use?" and that's exactly what I did. On this website, there will be 6 languages you can select from for now (more to come later). These languages will all have 3 different difficulties attached to them (Basic, Intermediate, and Advanced) allowing you to set your challenge. With around 50 flashcards on each page, you will see that you have plenty of material to learn from.
